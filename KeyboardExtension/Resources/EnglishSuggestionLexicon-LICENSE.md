# EnglishSuggestionLexicon.tsv

This suggestion lexicon is generated for MyCuKey from:

- The project's existing trusted common-word list and typing-domain seed words.
- `/usr/share/dict/web2`, the Webster's Second / FreeBSD English word list. Its local README states that the 1934 copyright has lapsed according to the supplier.

The numeric score column is a MyCuKey-local ranking hint, not a corpus frequency claim. It gives priority to project seed words and common words, then uses conservative length and morphology heuristics for the public-domain dictionary entries.
